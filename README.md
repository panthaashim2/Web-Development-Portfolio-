Web Fundamentals Portfolio

This project contains a multi-page portfolio created for the CET138 Web Fundamentals module. It demonstrates a working knowledge of full stack concepts, accessible user interfaces and interactive demos written with HTML, CSS, JavaScript and the Bootstrap 5 framework.
