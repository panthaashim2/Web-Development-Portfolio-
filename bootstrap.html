Bootstrap Framework

Bootstrap is one of the most popular front‑end toolkits for building responsive, mobile‑first websites. It includes prebuilt grid systems and UI components to speed up development.

What is Bootstrap?

Originally developed at Twitter, Bootstrap is a front‑end toolkit that combines CSS and JavaScript to help build interfaces quickly. Its responsive 12‑column grid system makes layouts adapt seamlessly across devices. A wide library of components—such as navbars, buttons, cards and modals—accelerates development and ensures consistent styling. Utility classes provide shortcuts for margins, padding, alignment, colours and more.

How It Works

The responsive grid uses .row and .col classes. On small screens each card takes up the full width (row-cols-1); on medium screens it switches to three columns (row-cols-md-3). Cards are built with the .card class and have no custom CSS here. The modal functionality is completely declarative—buttons include data-bs-toggle="modal" and data-bs-target attributes that link to a modal defined later in the page. Bootstrap’s JavaScript picks up these attributes and handles the rest.

The form demonstrates how Bootstrap’s validation styles work. By default the form has the novalidate attribute to prevent the browser’s native validation UI. When you click the “Validate” button JavaScript adds the was-validated class to the form, which triggers Bootstrap to apply is-valid or is-invalid classes based on each input’s validity state. Helper feedback elements display the success or error messages accordingly. This approach keeps your validation logic separate from your markup.

Key Takeaways

  • Bootstrap’s grid system makes responsive layouts straightforward using rows and columns.
  • Cards and modals are ready‑made components that require minimal configuration.
  • Utility classes simplify spacing, flexbox and colour adjustments.
  • Validation styles leverage the was-validated class to provide feedback without writing custom scripts.
Bootstrap Grid System
The grid system divides the page into 12 columns. You can combine columns to create flexible layouts and use breakpoint suffixes like -md or -lg to change widths on different devices. This ensures your design adapts elegantly from mobile phones to desktops.
Prebuilt Components
Bootstrap’s component library includes alerts, badges, buttons, dropdowns, navbars and more. These components are styled consistently and can be customised by adding or overriding classes. They speed up development and help maintain a cohesive look across pages.
Utility Classes
Utilities are single‑purpose helper classes that adjust padding (.p‑*), margin (.m‑*), flex properties (.d‑flex, .justify‑content‑*), colours and more. They allow you to tweak your layout without writing extra CSS.
