CSS: Styling & Layout

Cascading Style Sheets (CSS) describe how elements should be rendered on screen, in print or in other media. CSS controls the visual presentation and layout of web pages.

What is CSS?

CSS is a stylesheet language used to describe how HTML elements are presented. It allows you to control colours, fonts, spacing and layout independently of the document structure. By separating content from appearance, CSS makes your pages more maintainable and responsive. Modern CSS includes powerful features like custom properties (variables), flexbox, grid and media queries.

How It Works

The theme toggle changes the data-theme attribute on the <html> element. CSS variables (custom properties) are defined for colours and shadows in the :root selector and overridden under [data-theme="dark"]. When the attribute is toggled the entire palette updates. The layout toggle switches between CSS Grid and Flexbox by changing the container’s display property and related flex settings. The animation toggle adds or removes a class that applies a keyframe animation called bounce to each card. The box model demo uses nested elements with margin, border and padding to visualise how space is calculated. The line about specificity is styled by two selectors: a class in Bootstrap and an ID in our custom CSS – the ID wins because it is more specific.

A media query in the stylesheet adjusts padding and layout on screens narrower than 768 pixels. Try resizing the browser to observe how the spacing changes responsively.

Key Takeaways

  • CSS separates presentation from structure and uses selectors to target elements.
  • Custom properties (variables) make colour and spacing easy to manage.
  • Flexbox and Grid provide powerful tools for responsive layouts.
  • Specificity and the cascade determine which styles apply when selectors conflict.
  • Media queries adapt your design to different screen sizes.
