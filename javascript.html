JavaScript & Interactivity

JavaScript is a scripting language that enables you to create dynamic content, control multimedia, animate images and build rich web applications in the browser and on the server.

What is JavaScript?

JavaScript is the third layer of standard web technologies. HTML structures the content and CSS styles it, while JavaScript brings interactivity and logic. It runs inside the browser and, thanks to environments like Node.js, on the server as well. With JavaScript you can respond to user events, manipulate the DOM, perform calculations and make asynchronous requests to APIs. Modern features such as modules, promises and async/await make it suitable for building large applications.

Working Example: To‑Do List

This to‑do list demonstrates how JavaScript manipulates the DOM in response to user actions. You can add tasks, mark them complete, delete them and filter by status. Tasks are saved in localStorage, so your list will persist if you refresh the page.

Total: 0, Active: 0, Completed: 0

    No tasks. Add a task to get started!

    How It Works

    JavaScript listens for the form’s submit event, prevents the default page reload and reads the text input. The task is pushed into an array stored in localStorage and the UI is re‑rendered. Each list item has a checkbox to mark it complete and a button to delete it. Filter buttons set a state variable (all, active or completed) and re‑render the list accordingly. The counts are computed by filtering the tasks array. Because localStorage persists across sessions, tasks remain on subsequent visits.

    Key Takeaways

    • JavaScript brings interactivity to otherwise static HTML and CSS.
    • DOM manipulation involves selecting elements, responding to events and updating the document.
    • Storing data in localStorage enables persistence across page reloads.
    • State variables and rendering functions keep the UI in sync with data.
